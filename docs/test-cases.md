# LabelHub 数据标注平台 - 完整集成测试用例清单

> **文档说明**：本测试清单覆盖了 LabelHub 平台「数据生产 → AI 预审 → 人工审核 → 导出」的端到端全生命周期。包含功能链路测试、AI Agent 评测、RBAC 角色权限管控以及 Schema 版本管理的底层逻辑验证。

## 六、 非功能、安全 & 工程质量测试

| 测试ID | 测试场景 | 预期结果 |
| :--- | :--- | :--- |
| **TC-SEC-01** | **角色越权访问隔离** | 登录 Labeler 尝试访问 Owner URL / API，系统严格拦截 (403)，数据不泄露。 |
| **TC-QA-01** | TypeScript 类型安全 | 全栈核心模块（尤其是 Schema-core）无 `any` 滥用，Typecheck 0 error 编译通过。 |
| **TC-QA-02** | 核心纯函数单元测试 | `schema-core` 中的联动引擎、JSONPath 取值、版本校验拥有 Jest/Vitest 单测覆盖。 |
| **TC-QA-03** | **大表单渲染性能** | 采用局部渲染/依赖图算法，单页渲染 100+ 动态字段时输入无明显卡顿，保持 60FPS。 |
| **TC-QA-04** | 高并发抢单 / 幂等 | 多 Labeler 同时点击抢夺最后一个配额时，仅一人成功，不出现配额为负数的超卖。 |
| **TC-QA-05** | UI/UX 响应式适配 | 在 1280×800 (常规笔记本) 与 1920×1080 (外接屏幕) 下表单、侧边栏布局弹性铺开，不错乱。 |
